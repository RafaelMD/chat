# Chat

Esse código tem um chat que roda em python:
- Utiliza socketio para comunicação
- ChatGPT como IA para respostas
- ChatCompletion
- ChatGPT Functions
- Funções para Google Maps
- Funções para Google Search

## .env format
```
OPENAI_API_KEY= 
GOOGLE_MAPS_API_KEY= 
SEARCH_ENGINE_ID= 
```

## Rodar
```
python app.py
```
